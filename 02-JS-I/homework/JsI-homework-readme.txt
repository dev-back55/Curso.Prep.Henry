Conceptos




1.
Variable: es la forma de almacenar algo (valor) para usar más tarde. Podemos almacenar cadena de caracteres, números o valor booleano (verdadero o falso).


Strings: es una cadena de caracteres o bloque de texto, ej. “Hola soy Horacio” o “calle Tucuman 16”


Funciones (argumentos, return): son objetos invocables, es código que nos permite resolver algo repetitivo o cálculos. Argumentos, son los valores que le damos a la función para resolver y return es el valor que nos devuelve (resultado).
Declaraciones if: nos permiten comparar si algo es verdadero o falso y ejecutar el código correspondiente.
Valores booleanos (true, false): tienen dos significados verdadero o falso, se utilizan para comparar o saber si un valor es verdadero o falso.
